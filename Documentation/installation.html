Installation¶

To use the PY-Sports Discord bot, you need to set up the necessary dependencies and configurations on your system. This guide will walk you through the steps required to get the bot up and running.

Prerequisites¶

  1. Python: PY-Sports requires Python 3.7 or later to function correctly. If you don’t have Python installed on your system, you can download the latest version from the official Python website: https://www.python.org/downloads/

  2. pip: Pip is the package installer for Python. It should be installed automatically along with Python. You can verify its installation by opening a terminal/command prompt and running the following command:

    pip --version

    If you see the pip version information, you’re good to go. If not, you can install pip using the instructions provided in the Python documentation.

Installing Dependencies¶

  1. http.client, discord, python-dotenv: PY-Sports relies on several external libraries, including http.client for API communication, discord for interfacing with the Discord platform, and python-dotenv for handling environment variables.

    To install these dependencies, open a terminal/command prompt and navigate to the root directory of the PY-Sports project. Then, run the following command:

    pip install -r requirements.txt

    This command will read the requirements.txt file, which lists all the required packages and their versions, and automatically install them on your system.

Environment Configuration¶

  1. Create .env File: PY-Sports uses environment variables to store sensitive information, such as your Discord bot token and the API key for api-football. These variables are read from a file named .env in the root directory of the project.

    Create a new file named .env in the root directory and open it in a text editor.

  2. Add Environment Variables: Inside the .env file, add the following lines and replace the placeholders with your actual Discord bot token and api-football API key:

    DISCORD_BOT_TOKEN=your_discord_bot_token_here
API_FOOTBALL_KEY=your_api_football_key_here

    Save the .env file after adding the variables.

Running the Bot¶

With Python installed, dependencies installed, and the .env file set up with the required environment variables, you’re ready to run the PY-Sports Discord bot.

  1. Open a terminal/command prompt and navigate to the root directory of the PY-Sports project.

  2. Run the following command to start the bot:

    python bot.py

    Alternatively, if you have a start script like start.bat, you can use that to launch the bot.

The bot will now be active and ready to serve your Discord server with real-time sports statistics and information.

Congratulations! You have successfully installed the PY-Sports Discord bot and are all set to explore its exciting features. If you encounter any issues during the installation process, feel free to reach out for assistance.

If you have suggestions or want to contribute to the project, check out the “Contributing” section in the documentation for guidelines on how to get involved.

Enjoy using PY-Sports and have fun exploring sports data on Discord!
